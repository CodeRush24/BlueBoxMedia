
# BlueBox Media - YouTube Clone Web Application

BlueBox Media is a YouTube Clone Web Application


## Technology Used

 - ReactJS
 - Material UI
 - Rapid API

## Installation

Install with npm

```bash
  npm install 
```
```bash
  npm run dev 
```
Install with yarn

```bash
  yarn install 
```
```bash
  npm run dev 
```
